模塊介紹

重要 注意 提醒

Coding Style wiki​

基本說明​

代碼熱修復模塊，使用 HybridCLR (前身 Huatuo) 革命性的熱更新方案進行整合。

重要 如果相關建置完畢後，通過 MenuItem 點選 HybridCLR -> OxGFrame With HybirdCLR -> Complie And Copy To HotfixCollector，將會幫忙拷貝至 HotfixCollector 文件夾，再使用 YooAsset Collector 進行收集打包。

提醒 使用 HybridCLR 如有疑問時，請前往官方文檔進行熟悉。
