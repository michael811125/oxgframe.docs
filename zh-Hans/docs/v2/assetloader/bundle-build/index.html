資源構建

重要 注意 提醒

Coding Style wiki​

構建說明​

注意 請先詳見 YooAsset 資源配置 (蒐集)。

通過 MenuItem 點選 YooAsset -> AssetBundle Collector，開始配置資源 (蒐集)。

圖示說明：

  1. 提醒 顯示 Packages 視窗。
  2. 提醒 資源尋址名稱。

開始構建​

注意 另外可先詳見 YooAsset 資源構建 (打包)。

通過 MenuItem 點選 YooAsset -> AssetBundle Builder，開始選擇要構建的 Package 與 BuildPipeline，再選擇加密方式，最後點選 Click Build。

重要 切記不要更改 Build Version，以預設日期為主 (因為需要依照日期篩選出包)。

提醒 內置資源取決於 Copy Buildin File Option 選項，當 YooAsset 構建完畢後，將會依照設定放在 StreamingAssets 下。
