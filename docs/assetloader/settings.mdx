## 配置說明

了解全局配置文件。

![](/img/assetloader/img_1.png)

通過右鍵創建配置文件 (Project 窗體內右鍵 -> Create -> OxGFrame -> Create Settings -> Create Patch Setting)

![](/img/assetloader/img_2.png)

<Highlight color="#ff2f00">重要</Highlight> 請將配置文件放在 Resources 文件夾底下。

<Highlight color="#0373fc">注意</Highlight> 另外詳見 [YooAsset 全局配置](https://www.yooasset.com/docs/guide-editor/GlobalSettings)。
