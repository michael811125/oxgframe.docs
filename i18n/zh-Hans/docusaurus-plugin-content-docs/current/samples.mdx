## 额外说明

<Highlight color="#ff2f00">重要</Highlight> 示例工程会使用 **Unity New InputSystem**，所以建议先安装，否则会有 **EventSystem** 失效问题。

<Highlight color="#0373fc">New Project</Highlight> **[Input System](https://docs.unity3d.com/Packages/com.unity.inputsystem@1.8/manual/index.html)**

通过 Package Manager 安装 **(Install by name)**

```
com.unity.inputsystem
```
