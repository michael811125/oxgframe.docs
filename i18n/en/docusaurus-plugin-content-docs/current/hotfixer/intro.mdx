## Basic Introduction

The code hotfix module integrates the revolutionary hot update solution using HybridCLR (formerly Huatuo).

<Highlight color="#ff2f00">Important</Highlight> After completing the relevant build, clicking HybridCLR -> OxGFrame With HybridCLR -> Compile And Copy To HotfixCollector via MenuItem will help copy to the **HotfixCollector** folder, then use YooAsset Collector for collection and packaging.

<Highlight color="#00d679">Reminder</Highlight> If you have any questions about using [HybridCLR](https://github.com/focus-creative-games/hybridclr), please refer to the [official documentation](https://hybridclr.doc.code-philosophy.com/) for familiarization.
